# mysite
基于fastapi的个人网站、主要功能基于websocket的聊天应用  
现在正在制作中（前端采用原生的JavaScript以及Web API  
本项目没有打算使用框架，这似乎与这个框架满天飞的时代有些背离，但在未来会使用Typescript开发  

* 为什么不使用框架？
  - 1.这是一项个人项目，使用框架会增加额外负担。
  - 2.本人目前（2023，即将毕业的学生）依然在深入的学习JavaScript根本的知识，在开发过程中可以帮助学习更多原生的知识
  - 3.虽然本次没有使用框架，但个人对主流前端框架都有一定的了解（个人认为学好底层，才能更好的理解框架）
